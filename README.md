# Engineering Portfolio Website

Personal portfolio website built to showcase my work in CAD, coding, controls, electronics, and repair through project pages, technical write-ups, and visual case studies.

## About

This portfolio brings together the projects and skills I want to present in one place, with a focus on real-world engineering work, practical problem solving, and clear documentation.  
It is designed to show not just finished results, but also the process, constraints, and decisions behind each project.

## Featured Content

- CAD and design projects
- Programming and web-based tools
- Controls, electronics, and troubleshooting work
- Repair and reverse-engineering projects
- Technical case studies with images and project summaries

## Purpose

The goal of this site is to provide a structured, professional portfolio that makes it easy to review my technical background, project experience, and problem-solving approach across hardware and software.  
It also serves as a central place to connect my work, resume, and other professional links.

## Tech Stack

- HTML
- CSS
- JavaScript

This portfolio is an active project and will continue to grow as I add new work, refine project pages, and improve the overall presentation.
